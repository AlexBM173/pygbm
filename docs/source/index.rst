pygbm documentation
===================

Welcome to pygbm's documentation! This is a Python package for geometric Brownian motion simulations.

pygbm provides classes and tools for simulating and visualizing geometric Brownian motion paths,
which are commonly used in financial modeling and stochastic calculus.
